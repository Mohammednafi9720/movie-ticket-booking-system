# 🎬 Movie Ticket Booking System

A web-based Movie Ticket Booking System that allows users to browse movies, search films, select seats, and book tickets with an interactive UI.

---

## 🚀 Live Demo
https://your-username.github.io/movie-ticket-booking-system/

---

## 📌 Features
- 🎥 Display movies with posters, ratings, and price  
- 🔍 Search movies in real time  
- 💺 Interactive seat selection system  
- 🎟️ Ticket booking with validation  
- 📊 Admin dashboard with booking count  
- 🧠 Booking history using Linked List  
- 📱 Responsive UI design  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Data Structures:
  - Array (Movies storage)
  - 2D Array (Seat layout)
  - Linked List (Booking history)
  - Queue logic (booking flow)
